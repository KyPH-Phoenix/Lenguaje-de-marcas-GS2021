Apuntes de HTML5

<blockquote>

El blockquote sirve para citar una referencia de otro lugar. Lo siguiente es un blockquote de la definición de blockquote.

<blockquote cite="https://www.htmlquick.com/es/reference/tags/blockquote.html">
El elemento blockquote permite a los autores insertar citas en forma de bloques de contenido, usualmente compuestas por un párrafo, un grupo de párrafos o un conjunto de muchos otros elementos incluyendo imágenes (img), tablas (table) y artículos (article), entre otros. Esta es la principal diferencia entre este elemento y q, que está diseñado para citar únicamente líneas de texto.
</blockquote>

<address>

El "address" define una tarjeta de contacto del autor/editor de la página (o alguien, con carácter general).

<address>
Escrito por: <a href="mailto:ejemplomail@example.com">Hugo Notario.</a><br>
Visita mi página web <a href="https://lolesports.com/live/worlds/lvpes">Lolsito.</a><br>
C/ Cabana, nº43, Palma de Mallorca. </address>

Escrito por: Hugo Notario.
Visita mi página web Lolsito.
C/ Cabana, nº43, Palma de Mallorca.

<p>

Sirve para estructurar el texto por párrafos.

<Esto es un párrafo. De hecho casi todo en este documento son párrafos.>

Esto es un párrafo. De hecho casi todo en este documento son párrafos.

--------------------------------------------------
<span>

Los span son bloques de texto que pueden ponerse dentro de otros bloques (por ejemplo un <p>) y que nos permiten darle un estilo distinto a cierta parte del texto.

<p>Mi madre tiene los ojos <span style="color: brown">marrones</span> y las uñas pintadas de <span style="color: red">rojo</span>.<p>

Mi madre tiene los ojos marrones y las uñas pintadas de rojo.

--------------------------------------------------
<strong>

Strong sirve para indicar enfasis en el contenido del bloque.

<strong>Esto es Esto es un strong.</strong>

Esto es un strong.
--------------------------------------------------
<pre>

La etiqueta "pre" define texto preformatado

<pre>Texto preformatado.</pre>

Texto preformatado.
--------------------------------------------------
<code>

La etiqueta code sirve para definir un trozo de código.

<p>Podemos usar <code>while</code> para hacer un ciclo.</p>

Podemos usar while para hacer un ciclo.
